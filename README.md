# 🚀 Aditya Kumar Maurya — Personal Portfolio

A fully responsive, animated personal portfolio website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

🌐 **Live Site:** (https://levi-sage.github.io/PortFolio)

---

## ✨ Features

- **Custom animated cursor** with lagging ring effect
- **Scroll progress bar** at the top of the page
- **Typewriter effect** on the hero subtitle
- **Animated counters** for CGPA and project stats
- **Staggered scroll reveals** on every section
- **Parallax** hero name on scroll
- **Glitch effect** on name hover
- **Floating badges** and ambient orb backgrounds
- **Animated grid background** with noise grain overlay
- Fully **responsive** for mobile & desktop

## 🚀 Getting Started

### Run Locally

Just open the file in your browser — no build step needed:

```bash
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
open index.html   # macOS
# or double-click index.html on Windows/Linux
```

### Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Your site will be live at `https://your-username.github.io/my-portfolio`

### Preview in GitHub Codespaces

1. Click the green **Code** button → **Codespaces** → **New codespace**
2. In the terminal, run:
   ```bash
   npx serve .
   ```
3. Click **Open in Browser** when the port popup appears
